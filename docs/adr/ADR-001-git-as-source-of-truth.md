# ADR-001

## Título

Git como fuente de verdad.

## Estado

Aceptado.

## Decisión

Toda configuración persistente debe existir en Git.

No se aceptan cambios manuales que no estén reflejados en el repositorio.

## Consecuencias

- Infraestructura reproducible.
- Auditoría sencilla.
- Recuperación rápida.
